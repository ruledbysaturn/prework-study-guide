HTML

  • HTML, Hypertext Markup Language: is used to create structure of a webpage
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Declaration tells browser what document type it should expect when it is opened.
  • Child elements: elements that are nested inside other elements.
  • Parent element: element that holds all the child elements.

CSS

  • CSS, Cascading Style Sheets: defines colors, aesthetics, visual layout and organization of a webpage.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Three ways to style a webpage using CSS: inline CSS, internal CSS, & external CSS.
  • Separation of concerns: refers to separating code into different files based on use.
  • Do Not Repeat Yourself (DRY): coding principle to reuse/share pieces of code to reduce number of lines of code to be written or maintained.

Git

  • Git: version control system that lets you manage and keep track of each version of your project.
  • GitHub: cloud-based hosting service that allows you collaborate with others and store our code.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add: adds a change in the working directory to staging area.
  • Important to always do a git pull, to make sure you have the latest version of the project on local computer.

JavaScript

  • JavaScript: powerful programming language that can add interactivity to a website.
  • A Variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Array is a single variable used to hold a group of data, usually data that is related in some way.
  • String: sequence of characters used to represent text.
  • Boolean: true/false value. Used to decide which sections of code to execute.
  • Function: set of instructions that tells the computer how to perform a certain task.
